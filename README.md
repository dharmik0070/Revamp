## About The Project
>It is a mobile application that detects any object that the user wishes to remove/replace in the real world and gets the address and contacts of nearby dealers.

### Built With

* ##### Mobile Application
  - `Android`

* ##### Backend
  - ###### Server and core development
    - `Python`
  - ###### Database
    - `SQLite`
  - ###### Machine Learning
    - `Yolo Model`
